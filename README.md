# Portfolio - Ezequiel Fernández

Resumen  
Este repositorio contiene mi portfolio personal (frontend) construido con Vite + React + Tailwind (daisyUI). Incluye un layout base (Header con drawer accesible, MainLayout con Outlet, Footer), rutas centralizadas, theming con ThemeToggle y pruebas iniciales para la navegación móvil. Es un proyecto pensado para seguir creciendo con páginas reales, formularios y despliegue automático en GitHub.

Actualmente en desarrollo.
